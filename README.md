**Summary**

**MODULE 1:**
   Digital literacy is the ability to manage, access, understand, integrate, communicate, evaluate and create information safely and appropriately through digital technologies for employment, decent jobs and entrepreneurship.
   It discusses about various online tools like Cooltext, Prezi, Miro, Canva and GIF.

*Cooltext*
     Cooltext is a fun tool that helps create attractive and stylish text designs. It uses different fonts, shadows, colours and animations to make text look more eye-catching. It offers many styles like Neon text, Glitch text, 3D text, Handwritten fonts and Retro/Vintage text.
*Prezi*
     Prezi is a very creative presentation tool that works on a zoomable canvas instead of traditional slides. It makes presentations more engaging and memorable because you can zoom in and out to show connections between ideas. It has beautiful templates for party invitations, formal meetings, farewell events, etc. Its unique zoom feature keeps the audience interested and encourages creative thinking.
*Miro*
     Miro is an online whiteboard platform that makes brainstorming and planning very easy. You can add sticky notes, shapes, images and draw flowcharts and diagrams. It supports real-time collaboration, so multiple people can work together at the same time. It is perfect for group projects and visualising ideas.
*GIF*
     GIF (Graphical Interchange Format) is a short animated image that plays in a loop. It has no language barrier and is a great way to express emotions like happiness, excitement, sadness or nervousness. Popular platforms like GIPHY, Reddit and Tenor are used to find and share GIFs. I also learned about ezgif.com, a website to create or edit GIFs easily.
*Canva* 
   Is a user-friendly online tool that helps to create presentations, infographics and posters using ready-made templates. Due to its drag and drop feature it makes creation simple, fast and professional. I created a 5-step Digital Literacy Awareness Infographic using Canva template.
